HTML (Linguagem de Marcação de Hipertexto)

O que é HTML?

HTML é a abreviação para Linguagem de Marcação de Hipertexto (do inglês: HyperText Markup Language). É a linguagem de marcação padrão para criar e estruturar páginas web e aplicativos web. Com o HTML, podemos criar a estrutura de uma página web usando elementos de marcação que definem diferentes tipos de conteúdo, como texto, imagens, vídeos e muito mais.

História

O HTML foi criado por Tim Berners-Lee em 1991. Originalmente, HTML foi projetado como uma linguagem de marcação para cientistas compartilharem documentos. Com o tempo, HTML tornou-se uma das principais tecnologias da Internet, sendo fundamental para a criação de páginas web.

Como funciona?

O HTML é usado para estruturar conteúdo na web. Ele funciona através de 'tags' que são elementos marcados por sinais de menor (<) e maior (>). Estas tags são utilizadas para criar elementos como cabeçalhos, parágrafos, links, imagens e outros componentes que compõem uma página da internet.
